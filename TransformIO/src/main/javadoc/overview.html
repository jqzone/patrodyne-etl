TransformIO is an extract, transform and load tool driven by a XML configuration to specify record layouts using field level regex patterns.

The launcher for TransformIO is in the core package. The launcher transfers control to one of the user interface packages based on the launch mode. The mode determines which user interface will be invoked, as follows:

  • Graphical User Interface
  • Textual User Interface
  • Command Line Interface

The implementation for each interface is found in the corresponding sub-package.
